One-time setup required

The Bills module needs Supabase connected. This takes about 10 minutes and is free.

  1. 1
    Create a free project at supabase.com
  2. 2
    In the Supabase SQL Editor, run the file supabase/migrations/0001_init.sql from this repo
  3. 3
    In Supabase → Authentication → Providers, enable Google and paste your Google OAuth client ID + secret
  4. 4
    In Vercel project settings → Environment Variables, add:
    SUPABASE_URL, SUPABASE_ANON_KEY, SUPABASE_SERVICE_ROLE_KEY,
    ANTHROPIC_API_KEY, GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET
  5. 5
    Redeploy on Vercel — refresh this page
